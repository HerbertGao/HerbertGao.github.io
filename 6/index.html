【三秒兔】什么是CLASS10？浅谈存储卡容量外的属性

发表于 2015-03-15 更新于 2021-01-07 分类于 转载 本文字数： 1.9k 阅读时长 ≈ 2 分钟

这学期学校新开了一门课程叫摄影与摄像，课前老师让每位同学准备一张 8G 以上的 Class10 存储卡。关于存储卡，很多同学也许只听说过有容量尺寸以及品牌之分，关于 ClassX 这个新名词可能有些陌生。今天就来给大家科普一下存储卡除容量外的一些属性。

原载于三秒兔’S BLOG > 点击查看原文

一、关于尺寸

我们一般叫的存储卡其实就是指手机上所用的 MicroSD 卡（与 T-Flash 卡相互兼容），但存储卡其实有很多尺寸之分，我们常见的有 MMC 卡（老式诺基亚如我最早用的诺基亚 3230 就用的此卡，尺寸有多种）、SD 卡（我们常见的大卡 32mm24mm 多用于相机等设备）、T-Flash 卡/MicroSD 卡（市面上常见的小卡 15mm11mm 多用于手机等设备）。（除此之外还有很多如 MiniSD、记忆棒等由于日常生活用的较少在此不做介绍）

二、关于 ClassX

关于老师让买的 Class10 卡到底是个什么东西呢？其实这是除卡的容量大小外另一个重要的属性。SD 协会针对 TF 内存卡(SDHC)的传输规范：定义 ClassX 中的 X 为最低传输速度（单位：兆每秒）（并且最低传输速度一般指的是写入速度），并且此 X 的值为（X=2+2N）（N≥0），例：Class2 卡的最低写入速度为 2mb/s。所以 Class10 卡意味最低写入速度为 10mb/s 的卡。所以我们在购买存储卡时该如何选择 Class 值呢？一般来说我们所用的智能机中 MicroSD 卡大多为 Class4，已经可以满足日常数据存取需求。但对于单反相机这种设备我们就必须运用到 Class10 以及更高的存储卡，单反照片一般来说一张有 8mb 左右（根据相机设置的参数不容照片大小会有不同变化），所以为了满足拍摄需求必须选择这种高速率的存储卡。总而言之，Class 后的数字越高，就可以满足更高速率要求的存储需要。

既然了解了存储卡的 Class 属性，那我们在购买存储卡时该如何分辨我购买到的到底是 Class 几的卡呢？很简单，标准的 SD/TF（MSD）卡上均统一标有该属性，大家可以在存数卡正面找到一个 C 中间带着一个数字的标志，C 中间的数字为 4 即该卡为 Class4 卡，以此类推，C 中间数字为 10 即为 Class10 卡。

当然上面那种辨别方法只是最直观的分辨方法，实际操作中考虑到个别商家卖假卡需标的情况我们还需将其插入电脑中测试其速率才能判断该卡到底是多少速率的 SD 卡，测试过程中请直接将 SD 卡插入笔记本自带的 SD 卡卡槽测试，在空卡中存入一个大于 500mb 的文件查看写入速度即可分辨，注意：利用 USB 读卡器和数据线方式测试均不能得到可靠数据（比真实数据偏低）。

三、其他事项

目前市面上的存储卡除了标有容量和 CX 之外还有一些数据可能大家还不是很理解：

比如我买的 SD 卡为什么会写有 SDHC？
SDHC 是”Secure Digital High Capacity“的缩写，即“高容量 SD 存储卡”。2006 年 5 月 SD 协会发布了最新版的 SD 2.0 的系统规范，在其中规定 SDHC 是符合新的规范、且容量大于 2GB 小于等于 32GB 的 SD 卡。

我买的存储卡上没有标有 C10 却有一个 U1 这是为什么？有的存储卡上为何标有罗马数字 Ⅰ？
这两个标识分别是指 UHS-I 和 UHS Speed Class 1。其中用大写字母“I”表示的就是 UHS-I 标准。基于这种标准的 SDXC/SDHC 卡支持 UHS（Ultra High Speed，超高速）接口，接口带宽达到 104MB/s（不代表实际传输速度），并向下兼容原有的 25MB/s 接口。在此基础上，如果 SDXC UHS-I 或 SDHC UHS-I 存储卡性能能够满足实时存储高清视频的标准，即可标识“UHS Speed Class 1”。所以 U1 卡将远优于 C10 卡，所以对于需要使用单反设备的同学选择该卡是一个更好的选择。

我该如何在网上购买到好一点的存储卡？
关于这个问题是没有绝对的答案的，除了考虑到上面讲到的两个参数之外，品牌和版本都是需要纳入考虑的。据博主个人的经验来说网购卡还需在正品旗舰店购买，关于品牌闪迪和三星无异于是一个很好的选择，虽然说金士顿也在存储界数一数二但是市面上假货太多不容易买到官方正品所以还是选择前面两个品牌较好。如果你选择了闪迪的内存卡，红金 ultra 版本一般来说是个很好的选择，如果你追求极致可以选择红金 Extreme 或者 E Pro 版本.（Ultra / Extreme /Extreme Pro 的速度等级不同，Ultra 高速卡，Extreme 是极速卡，Extreme Pro 是超级卡。）
