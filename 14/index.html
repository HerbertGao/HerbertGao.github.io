美区Apple ID，从被封到解封

发表于 2018-09-26 更新于 2019-08-15 分类于 原创
本文字数： 1.1k 阅读时长 ≈ 2 分钟

注册美区 Apple ID 之后，事儿几乎就没少过。
从 iTunes 礼品卡支付，到绑定美区 PayPal，到多次被禁止支付，再到这次 Apple ID 被封……

Apple ID 被封

昨晚手机还一直在好好用，今天想下载一个应用的时候，App Store 突然提示 Your Apple ID has been disabled.

在 Apple 官网查询可能是账号被别人登录连续输错密码导致 Apple ID 被锁，所以我先尝试了修改 Apple ID 密码，但很快我发觉实际不是这个导致的问题，因为如果是被锁的话，我自己应该也无法登陆才对。

密码修改之后，问题并没有解决，还导致我必须新构思了一个以前没用过的密码。

致电 Apple 技术支持

由于 V2EX 上有人有过类似的经历，说通过 Apple 官网 (https://getsupport.apple.com) 提交技术支持请求，致电 Apple 技术支持可以解决问题，即使是中国区官网也可以。我在官网提交了技术支持请求，过了一会，就有 021 的苹果中国的电话打了过来。

接通之后按 1，转人工服务，首先是第一级客服，了解完问题后转到了账单部门的客服，然而账单部门客服给了一些解决方案完全没什么作用（比如退出 App Store 的 Apple ID 再重新登录，找个 Apple 官方提供的应用来下载之类的）。

实际上在这次致电 Apple 技术支持的时候，我犯了个错，没有第一时间告知客服我的 Apple ID 是美国账号，实际上如果先告知了时美国账号，后续他们就会采取不同的策略来解决问题了。

折腾了二三十分钟，通话忽然断掉了，然而问题还没有解决，我只好通过 Apple 官网再次致电。

问题解决

还是和上面一样的步骤，再次致电 Apple 的时候，第一级客服注意到了我的 Apple ID 是 Gmail 邮箱，询问了我是否是中国区的账号，我告知是美国区的，他说他的权限无法查询我的账号，所以需要把我的问题转交给别的部门。

听了一会音乐后，通话转给了一个台湾腔的小姐姐，小姐姐告知需要执行重启我的账号的操作。

小姐姐指导我登录了 Apple ID 的网站(https://appleid.apple.com)，登录之后，在页面最下方有一个「支持 PIN」，获取之后提交给了她。

再稍微过了一会，她告知 Apple ID 已经可以使用了，尝试了一下，问题果然已经得到解决了！

总结

如果你出现了相同的问题，如果发现不是密码的问题的话，直接去找客服，告知是美区账号，然后执行相同的操作即可，就不用像我这样浪费时间了，哈哈！
